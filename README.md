#### Purpose: To teach fundamental Java Libraries and how to use them in a project. 

#### Java Libraries used in this workshop:
* FileReader
* BufferedReader
* Writer
* PrintWriter
* GUI Components
  * JFrame
  * JPanel
  * JLabel
  * JButton
  * JTextField
* Event Listeners
  * MouseListener
  * MouseMotionListener
  * ActionListener
  * KeyListener
* Layouts
  * FlowLayout
  * GridLayout
  * GridBagLayout
